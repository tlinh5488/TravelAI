🔧 Backend API Endpoints

Base URL: http://localhost/TravelAI/backend/api/

📍 Locations API

  • GET /locations.php - Lấy danh sách địa điểm
  • ?category=beach - Lọc theo loại
  • ?search=bãi - Tìm kiếm

📅 Itinerary API

  • GET /itinerary.php - Lấy lịch trình
  • POST /itinerary.php - Tạo lịch trình mới
  • PUT /itinerary.php?id={id} - Cập nhật lịch trình
  • DELETE /itinerary.php?id={id} - Xóa lịch trình

💬 Chat API

  • POST /chat.php - Gửi tin nhắn đến AI
  • GET /chat.php - Lấy lịch sử chat

🛣️ Route API

  • GET /route.php?from={id}&to={id}&transport={type} - Tìm tuyến đường

🗄️ Database Schema

Database Name: travel_ai

Các bảng chính:

  • users - Thông tin người dùng
  • locations - Địa điểm du lịch
  • itineraries - Lịch trình
  • itinerary_items - Chi tiết lịch trình
  • chat_messages - Lịch sử chat
  • favorites - Địa điểm yêu thích
  • routes - Tuyến đường

🚀 Hướng Dẫn Cài Đặt Nhanh

1️⃣ Yêu Cầu Hệ Thống

  • XAMPP hoặc Apache + PHP + MySQL
  • PHP 7.4+ và MySQL 5.7+
  • Trình duyệt hiện đại

2️⃣ Copy Dự Án

C:\xampp\htdocs\TravelAI\ (hoặc /var/www/html/TravelAI/)

3️⃣ Tạo Database

  • Mở phpMyAdmin: http://localhost/phpmyadmin
  • Import file: database/schema.sql
  • Database tên: travel_ai

4️⃣ Start Server

  • XAMPP Control Panel: Start Apache + MySQL

5️⃣ Truy Cập

http://localhost/TravelAI/frontend/index.html
